# Weather Dashboard

A simple weather web app that fetches real-time weather data for any city using the OpenWeatherMap API.

---

## Live Demo

[weatherdashboard-taupe.vercel.app](https://weatherdashboard-taupe.vercel.app/)

---

## Tech Stack

- React
- OpenWeatherMap API
- CSS
- Vercel

---

## Features

- Search weather by city name
- Displays temperature, feels like, humidity and wind speed
- Press Enter or click Search to fetch results
- Error handling for invalid city names

---

## How It Works

1. User enters a city name
2. App sends a request to OpenWeatherMap API
3. Response data is displayed as a weather card

---

## Run Locally

```bash
git clone https://github.com/shyam6767/weatherdashboard.git
cd weatherdashboard
npm install
npm start
```

Replace the API key in `App.js` with your own from [openweathermap.org](https://openweathermap.org).
